# 2. 使用和调用

## 2.1 通过 python -m pytest 调用pytest

可以通过Python解释器从命令行调用test

``python -m pytest [...]``

## 2.2 获取版本、选项名称、环境变量的帮助信息

	pytest --version # 展示Pytest的导入路径
	pytest --fixtures # 展示可用的内置函数参数
	pytest -h | --help # 展示命令行帮助信息和配置文件选项
